- screens/
	- LoginScreen.js
	- HomeFeedScreen.js
	- RecipeDetailScreen.js
	- AIGeneratorScreen.js
	- MealPlanScreen.js
	- GroceryScreen.js
	- ProfileScreen.js
- components/
	- RecipeCard.js
	- IngredientList.js
	- VideoPlayer.js
	- GroceryItem.js
- services/api.js

## Example Screens
- Login: username/password, login button
- Home Feed: scrollable recipe cards
- Recipe Detail: recipe info, video player
- AI Generator: ingredient input, generated recipe
- Meal Plan: weekly meal plan
- Grocery: categorized grocery list, check-off
- Profile: username, saved recipes, subscription

## Instructions to Run
1. Install Expo CLI: `npm install -g expo-cli`
2. Navigate to /mobile-app
3. Install dependencies: `npm install`
4. Start app: `expo start`
5. Scan QR code with Expo Go app (iOS/Android)
