# Development Guide

## Setup
1. Clone repository
2. Install dependencies: `npm install`
3. Configure environment variables: `.env.example`
4. Start services: `docker-compose up --build`
5. Start local dev server: `npm run dev` in /apps/dashboard

## Contributing
- Fork repo
- Create feature branch
- Submit pull request

## Testing
- Run tests: `npm test`

## Troubleshooting
- See logs in each service
- Check .env config
- Use `docker ps` to verify containers
