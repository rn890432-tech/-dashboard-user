# SOC Dashboard & Email Analytics Wireframes

## SOC Dashboard
- Heatmap of cyber attack events (region/time aggregation)
- List of recent SOC alerts (severity, status, description)
- Threat details panel (drill-down from heatmap)
- Manual override (KillSwitch)
- User activity and operational users

## Email Analytics
- Deliverability stats (sent, delivered, bounced, complaints)
- Engagement metrics (open/click rates, device, geo)
- Suppression/compliance overview
- Email log search/filter
- Unsubscribe preferences management

---

## Example Layout

| SOC Dashboard | Email Analytics |
|--------------|----------------|
| Heatmap      | Deliverability |
| Alerts List  | Engagement     |
| Threat Panel | Suppression    |
| KillSwitch   | Log Search     |
| User Panel   | Preferences    |
